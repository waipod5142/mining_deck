# GEOID × Streamax — Smart Mining Solution for Thailand

An interactive HTML slide deck presenting a smart-mining solution for the Thai mining
sector, combining **Streamax** mining AI hardware with **GEOID (Thailand)** local
telematics, integration and support. Prepared for a Technology 4.0 mining-productivity
seminar (DPIM).

## 🔗 Live presentation

Once GitHub Pages is enabled (Settings → Pages → Deploy from a branch → `main` / root):

**https://waipod5142.github.io/mining_deck/**

## ▶️ How to present

- Open the page in any modern browser (best viewed full-screen / presentation mode).
- Navigate with the **← / →** arrow keys, the on-screen **‹ ›** buttons, or **swipe** on touch devices.
- On the **feature showcase** slide, ← / → step through all nine Streamax modules one by one,
  or click a number (1–9) to jump to a module.

## 🗂️ Slides

1. Title

9. National alignment — Thailand's Mining 4.0 / DPIM agenda
10. Call to action

## 📝 Notes

- The deck loads Chart.js and the Streamax product demo videos from the web, so an
  **internet connection** is required when presenting. If a video clip is unavailable,
  an animated fallback graphic keeps the panel active.
- All ROI/benefit figures are framed as **illustrative or to-be-measured in a pilot** —
  no financial benefit is claimed up front.
